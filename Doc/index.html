Navbar

Navbar is a fixed element act like a header. Its contain site title, main navigation and search navigation. There are two types navbar on this template, standard navbar and another one navbar that contain tabs.

Standard navbar use class fixed-navbar on page (id) element, here the code. 
<div id="page" class="fixed-navbar">
	<div class="top-navbar">
		...
	</div>
	...
Tabs navbar use class fixed-tabs on page (id) element, also its added with-tabs class inside top-navbar element, here the code. 
<div id="page" class="fixed-tabs">
	<div class="top-navbar with-tabs">
		...
	</div>
	...

Button trigger

Button trigger used to show left sidebar / sidenav where by default is hidden. You can use button <btn ... or navigation link <a ... . Here the code.

<a href="#" data-target="slide-out-left" class="sidenav-trigger">

</a>

CSS Files & Structure

We provide 2 css files which is actually have same function. There are development version (style.css) and production version (style.min.css). These files located in assets/css/ directory.

We store all additional styles including plugins style, template font, and material icons to this style.css with import mechanism. So in every HTML template, just only 1 referring file, style.css or style.min.css

HTML template style/css referring file

...
<link rel="stylesheet" type="text/css" href="assets/css/style.css">
...

	...

	/* Fixed navbar -------------------------------*/
	if( $('.fixed-navbar').length ) {
		$(window).scroll(function () {
			if ($(this).scrollTop() > 0 ) {
				$('.fixed-navbar .top-navbar').addClass("fly");
				$('.fixed-navbar .content-container').css('margin-top', '54px');
			} else {
				$('.fixed-navbar .top-navbar').removeClass("fly");
				$('.fixed-navbar .content-container').css('margin-top', '0');
			}
		});
	}

	...

		var fn = $('.front-navigation').outerHeight(true),

Scroll to top

Scroll to top navigation is element navigation used for user to make easy back to top. This element put on footer section, and here the HTML tag. 
<div id="to-top" class="main-bg">
	<span class="ti-arrow-up"></span>
</div>
And here jQuery script. 
$(function() {
	
	....

	/* ==============================================
	** Scroll top
	 ==============================================*/
	var winScroll = $(window).scrollTop();
	if (winScroll > 1) {
		$('#to-top').css({bottom:"0"});
	} else {
		$('#to-top').css({bottom:"0"});
	}
	$(window).on("scroll",function(){
		winScroll = $(window).scrollTop();

		if (winScroll > 1) {
			$('#to-top').css({opacity:1,bottom:"0"});
		} else {
			$('#to-top').css({opacity:0,bottom:"-30px"});
		}
	});
	$('#to-top').on("click", function() {
		$('html, body').animate({scrollTop: '0px'}, 800);
		return false;
	});
	/*------------------------------------------------------*/

	...

});
